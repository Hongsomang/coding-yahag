### HTML실습

1.모델링을 html로 만들기

+ 제목를 나타 낼 때
  + ```<h1>코딩 야학<h1>```
+ 소제목
  + ```<h2>변수와 상수</h2>```

2.이미론적인 웹

+ Semantic Web -의미가 잘 드러나는 웹
+ 네비게이션 바
  + ```<nav> </nav>```
+ 제목
  +  ```<header><h1>코딩야학</h1></header>```
+ 본문
  + ```<article></article>```
